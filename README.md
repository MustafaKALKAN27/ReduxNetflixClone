# Proje NetflixClone

-Aşağıda projenin kurulum aşamaları belirtilmiştir. 
-Projede ReduxThunk kullanılmıştır.
-Tmdb üye olunarak api için kişisel token alınmıştır.
-Alınan Token api isteği atmak için kullanılmıştır.

# Projeyi Kurmama
 
  -npm create vite .
  -npm i

# Kütüphaneler

- react-router-dom
- redux
- react-redux
- redux-thunk
- axios
- millify
- tailwind
